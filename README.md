# Matrix High School Website

This is a school website project created for Matrix High School, Sikar.
The website is designed using HTML and CSS.

## About
This website shows information about the school, including:
- School facilities
- Students achievements
- Campus image
- Official website link

## Technologies Used
- HTML
- CSS

## How to Use
Open the index.html file in any web browser to view the website.
